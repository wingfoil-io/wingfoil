WINGFOIL v2.1.1 — March 2026
Rust stream processing for high-performance DAG-based pipelines.

——— WHAT'S NEW ———

  KDB+ adapter (production-ready)
  Connect your streams directly to KDB+ time-series databases with full
  symbol interning. Built for the latency and throughput demands of
  real-time financial data.

  Pandas integration (wingfoil-python)
  Seamless Python interop — pass stream output directly into pandas
  DataFrames.

  Time-based windowing
  New windowing nodes with Python examples included.

  Code quality sweep
  Improvements across the board to consistency and correctness.
